Objetivo General

Permitir que los estudiantes del curso Diseños de Sitios Web accedan a material de libre uso, actualizado, pertinente y directamente relacionado con la temática que aborda el curso en formato de lectura y audiovisual.

Lograr un entorno de desarrollo online de manera colaborativa haciendo uso de las herramientas Git Bash y Github Desktop

Detalles del Curso

El curso que te presentamos, titulado “Diseño de Sitios Web” y creado por el estudiante Jhon Botello, es de los pocos de este perfil que abarca lenguajes y tecnologías tan actuales como son el HTML5 y el CSS3.

Si quieres estar realmente actualizado, debes saber los nuevos lenguajes con los que se diseña la web, es decir HTML5 y CSS3, pero esto no es solo conocer las nuevas etiquetas, implica mucho más. En este curso entenderás cómo y cuándo utilizar cada lenguaje.

Contenido del Curso :

  1. HTML5 + CSS3 (Desarrollo de un Proyecto Real)
  2. Responsive Design y Mobile First
  3. Posicionamiento SEO
  4. Control de Versiones con Git y GitHub
  5. Diseño y Usabilidad
  6. Animaciones CSS
